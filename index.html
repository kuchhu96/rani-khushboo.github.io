About

I am a Postdoctoral Associate at Virginia Tech. My research focuses on computer architecture, memory systems, and CXL-based disaggregated memory.

I work on architectural simulation using gem5 and study performance, correctness, and security challenges in modern computing systems.

My broader interests include scalable memory architectures, cache coherence, system-level evaluation, and secure hardware-software mechanisms.

CXL-Based Disaggregated Memory Systems

Architectural Simulation with gem5
