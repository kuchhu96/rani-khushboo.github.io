About

I am a researcher focused on secure computer architecture, processor design, memory systems, and architectural support for software and hardware security. My work spans compartmentalization, memory safety, microarchitectural defenses, and dependable computing for modern systems.

I am interested in building deployable and practical mechanisms that strengthen execution isolation, protect data, and improve resilience in performance-critical computing environments.
